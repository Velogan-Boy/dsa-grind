# DSA Grind

This repository contains my solutions to various **Data Structures and Algorithms problems** that I solved while practicing for coding interviews and improving my problem-solving skills.

I created this repo mainly to track my progress and keep all my solutions in one place so I can revisit them later. Most of the problems are from platforms like **LeetCode** and other common interview preparation resources.

The main goal is simple: **practice consistently, understand patterns, and improve problem-solving ability.**

---

## What you'll find here

The repository includes problems from different DSA topics such as:

- Arrays
- Linked Lists
- Stacks & Queues
- Trees & Binary Trees
- Graphs
- Recursion & Backtracking
- Dynamic Programming
- Sliding Window
- Two Pointers
- Binary Search
- Greedy Algorithms
- Other common interview patterns

Most solutions are written with clarity in mind so that they are easy to revisit and understand later.

---

## Why I created this repo

When practicing DSA regularly, it’s easy to forget patterns and approaches after some time.  
This repository helps me to:

- Track the problems I’ve solved
- Revisit tricky problems
- Recognize patterns across problems
- Stay consistent with daily practice

You can think of this repo as my **personal DSA practice log**.

---

Problems are organized into folders based on topic or pattern to make navigation easier.

Example structure:

Instead of solving random problems endlessly, I focus on:

1. Understanding the underlying pattern
2. Solving a few variations

This approach helps build stronger intuition for technical interviews.
